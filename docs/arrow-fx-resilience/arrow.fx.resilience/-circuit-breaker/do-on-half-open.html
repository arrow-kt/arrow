doOnHalfOpen

fun doOnHalfOpen(callback: suspend () -> Unit): CircuitBreaker(source)

Returns a new circuit breaker that wraps the state of the source and that will fire the given callback upon the circuit breaker transitioning to the CircuitBreaker.State.HalfOpen state.

It is useful for gathering stats.

NOTE: calling this method multiple times will create a circuit breaker that will call multiple callbacks, thus the callback given is cumulative with other specified callbacks.

Return

a new circuit breaker wrapping the state of the source

Parameters

callback

is to be executed when the state evolves into CircuitBreaker.State.HalfOpen
