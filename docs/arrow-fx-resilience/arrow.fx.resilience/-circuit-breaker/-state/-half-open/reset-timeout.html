Parameters

resetTimeout

is the current reset timeout that the CircuitBreaker has to stay in Open state. When the reset timeout lapsed, than the CircuitBreaker will allow a test request to go through in HalfOpen. If the test request failed, the CircuitBreaker will go back into Open and it'll multiply the resetTimeout with the the exponential backoff factor.
